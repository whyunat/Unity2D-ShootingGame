# Unity2D-ShootingGame
## 프로젝트 개요
**개발 기간**: 2025.03.17~2025.03.31 (팀프로젝트, 5명)  
**역할**: 개발 & 스테이지 배경 제작  
**기술 스택**: [C#](https://img.shields.io/badge/C%23-239120?style=for-the-badge&logo=c-sharp&logoColor=white), [Unity](https://img.shields.io/badge/Unity-100000?style=for-the-badge&logo=unity&logoColor=white)  
**유니티버전**: Unity 6(6000.0.40f1)

## 주요 기여도
- Tilemap을 활용하여 1,4 스테이지 배경 제작
- 씬 이동 포탈
- 버그 수정
